# Project Setup

## Getting Started

1. Open a terminal in the project folder.
2. Install dependencies:
   ```bash
   npm i
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

The app will now be running — you're project should now be opening in another tab (make sure to not block popups).

Context
You have a simple e-commerce app that:
Fetches products from ProductService.cs
Displays the products in a list
Task
Your task is to implement a filter panel for the product list.
An API endpoint is available in ProductController.cs at:POST /api/products/filter
Timebox: 1–2 hours
Deliverable: Commit your code to the repository and open a pull request.
Requirements
Ability to filter products with lime flavour only.
Ability to filter products that are beverages only.
Technical interview
During the technical interview, we’ll review your pull request together and discuss:
Your approach and implementation of the filter functionality
Ideas for improving and extending the existing codebase
Be prepared to explain your design decisions and share how you would enhance the overall structure, performance, or maintainability of the application.
